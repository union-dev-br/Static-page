Política de Privacidade - Aplicativo Economize

Última atualização: 14 de Abril de 2025

1. Introdução

O aplicativo Economize é um software de gestão financeira pessoal que funciona totalmente offline, desenvolvido com foco na privacidade e segurança dos usuários.

2. Coleta e Uso de Informações

2.1 Informações Armazenadas Localmente

Todos os dados inseridos no aplicativo (orçamentos, despesas, receitas, metas financeiras e histórico de preços) são armazenados exclusivamente no dispositivo do usuário.

Não realizamos nenhuma coleta, transmissão ou sincronização de dados com servidores externos.

2.2 Permissões do Aplicativo

O aplicativo solicita apenas as permissões mínimas necessárias para seu funcionamento:

  • Armazenamento: para salvar dados localmente no dispositivo.
  • Captura de tela: para funcionalidade de exportação de relatórios (se aplicável - verifique se o app realmente pede essa permissão).

3. Compartilhamento de Dados

  • O aplicativo não compartilha dados com terceiros.
  • Não existe funcionalidade de login ou sincronização.
  • Não coletamos dados de uso ou análise.
  • Não utilizamos rastreadores ou ferramentas de analytics.

4. Segurança dos Dados

  • Todos os dados são armazenados localmente utilizando SQLite (ou a tecnologia de banco de dados local do seu app).
  • O acesso aos dados é restrito ao próprio aplicativo no dispositivo do usuário.
  • A exclusão do aplicativo remove permanentemente todos os dados armazenados.

5. Dados de Crianças

Não coletamos intencionalmente dados de crianças menores de 13 anos, uma vez que o aplicativo não coleta nenhum dado pessoal.

6. Alterações na Política de Privacidade

Reservamo-nos o direito de atualizar nossa política de privacidade periodicamente. Alterações significativas serão comunicadas através de atualizações do aplicativo ou aviso dentro do próprio app.

7. Contato

Para questões relacionadas a esta política de privacidade, entre em contato através de contato@union.dev.br.

8. Exclusão de Dados

Para excluir todos os dados do aplicativo, o usuário pode:

  • Utilizar a função "Limpar Dados" ou "Redefinir App" nas configurações do aplicativo (se disponível).
  • Desinstalar o aplicativo do dispositivo.

Esta política de privacidade foi elaborada para demonstrar nosso compromisso com a privacidade dos usuários e explicar que o aplicativo Economize opera exclusivamente offline, sem coleta ou transmissão de dados pessoais.
